The .zip file here contains midi files for 46 excerpts from the workbook
and instructor's manual for _Tonal Harmony_ by Stefan Kostka and
Dorothy Payne. 
 
Each excerpt has been annotated with midi lyric events showing the harmonies for the excerpt. 
Chord names are assumed to correspond to the segment starting where they occur in the file up to where the next chord name appears.
 
The chord names are in a modified “Jazz” style notation. An example translation from a Roman numeral analysis is as follows…
 
C: I-vii06-I6 becomes C maj 0 - B dim 1 - C maj 1. 
 
Here the number following the chord name corresponds to inversion 0 = root position, 1 = 1st inv, 2 = 2nd inversion, etc.
 
Neopolitan chords simply get the name of the chord they would have if they were taken out of context. Therefore, in C, the flat II chord would just be called Db maj. 
 
German, Italian and French 6th chords are labeled with an “X” for the root, an inversion of 0 and the chord quality. Thus, “X ger6 0” would be an example. This is due to the lack of an “official root” for these chords.
